## Descrição do Projeto
Este projeto demonstra os conceitos de Programação Orientada a Objetos em Java, focando na aplicação prática de herança, sobrescrita de métodos (`@Override`) e polimorfismo. 

O tema escolhido foi a modelagem de componentes de hardware para a montagem de um setup de computador. Criei uma superclasse abstrata no conceito chamada `Componente`, que concentra atributos universais como marca, modelo e preço. A partir dela, derivei duas subclasses: `Processador` e `PlacaDeVideo`, cada uma com suas especificações técnicas únicas. O polimorfismo é evidenciado no método de realizar *benchmarks*, onde o sistema consegue testar as peças de forma dinâmica na classe `Main`.
